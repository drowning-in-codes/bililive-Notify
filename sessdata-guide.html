如何找到 SESSDATA

SESSDATA 是你登录 B 站后浏览器 Cookie 中的一个值。按照下面步骤即可找到并复制。

Chrome / Edge 获取步骤

  1. 先在浏览器中登录 B 站账号。
  2. 按 F12 打开开发者工具。
  3. 切换到 Application（Edge 同名）。
  4. 左侧打开 Storage > Cookies，选择 b站域名。
  5. 在列表中找到 SESSDATA，复制对应的 Value 值。
  6. 回到登录窗口，粘贴到 SESSDATA 输入框并点击登录。

Firefox 获取步骤

  1. 登录 B 站后按 F12。
  2. 打开 存储（Storage）面板。
  3. 选择 Cookie，找到 b站域名。
  4. 搜索并复制 SESSDATA 的值。

请勿将 SESSDATA 分享给他人。它等同于账号登录凭证，泄露后可能导致账号风险。
